# Calculator-Application
A responsive calculator web application developed with HTML, CSS, and JavaScript, featuring basic arithmetic operations and a clean user interface.
A simple, clean, and responsive calculator web application built using HTML, CSS, and JavaScript. This project demonstrates core frontend development concepts such as UI design, responsiveness, and JavaScript event handling.

Features

Supports basic arithmetic operations: addition, subtraction, multiplication, and division

Displays the expression before evaluation

Modern UI with a clean color theme

Fully responsive design for mobile, tablet, and desktop screens

Interactive buttons with hover and click effects

Clear button to reset calculations

Technologies Used

HTML5 for structure

CSS3 for styling and responsive layout

JavaScript (Vanilla JS) for calculator logic and interactivity

How It Works
The user clicks number and operator buttons to build an expression. When the equals button is pressed, the expression is evaluated and the result is displayed instantly. The clear button resets the calculator to its initial state.
